# NeuralOps Landing Page

A premium AI SaaS landing page developed as part of the **Next-Gen AI Platform Speed Run** challenge. The project focuses on creating a modern, responsive, and high-performance landing page that combines clean UI design with efficient client-side architecture, accessibility, and SEO best practices.

The application demonstrates dynamic pricing computation, responsive layout transformations, semantic HTML structure, and optimized native animations without relying on external UI component libraries.

# Project Overview

NeuralOps is designed as a modern landing page for an AI-powered data automation platform. The objective was to create a visually appealing, conversion-focused interface while implementing efficient frontend architecture and responsive behavior.

The project emphasizes:

- Responsive UI across desktop, tablet, and mobile devices
- Dynamic pricing calculations
- Optimized DOM updates
- Native CSS animations
- Accessible user interactions
- Performance-oriented implementation
- SEO-friendly semantic HTML

---

# Features

## Hero Section

- Modern SaaS-style hero banner
- Responsive navigation
- Call-to-action buttons
- Product preview panel
- Responsive typography

## Dynamic Pricing Engine

Implements a pricing system capable of:

- Monthly billing
- Annual billing
- INR pricing
- USD pricing
- EUR pricing
- Dynamic price computation
- Localized currency symbols
- State-isolated DOM updates

## Responsive Bento Grid

Desktop experience features a Bento Grid interface that automatically transforms into an Accordion layout on mobile devices.

The implementation preserves the currently active card while switching between layouts to ensure a seamless user experience.

## Scroll Animations

- Native CSS transitions
- Intersection Observer API
- Smooth reveal animations
- Performance-optimized effects

## Responsive Design

The interface adapts smoothly across:

- Desktop
- Laptop
- Tablet
- Mobile

without layout breaking or horizontal overflow.

## Accessibility

The project follows accessibility best practices including:

- Semantic HTML5
- ARIA labels
- Accessible buttons
- Keyboard-friendly interactions
- Proper document hierarchy

## SEO

The page includes:

- Meta description
- Open Graph tags
- Semantic HTML structure
- Crawlable content
- Optimized page metadata

# Running the Project

Clone the repository

```bash
git clone https://github.com/aymn10/neuralops-landing.git
```

Navigate into the project

```bash
cd neuralops-landing
```

Open the application

```
index.html
```

No installation or package manager is required since the project is built using Vanilla HTML, CSS, and JavaScript.
